# Social share image (Open Graph / Twitter)

- **og-image.svg** — Image used when the site is shared (slogan + MalangaCraft). Used in meta `og:image` and `twitter:image`.
- For best support on Facebook and Twitter (they often require PNG/JPEG), export this SVG to **1200×630 PNG**, save as `og-image.png` in `public/`, and in `index.html` change the meta `og:image` and `twitter:image` URLs from `og-image.svg` to `og-image.png`.
